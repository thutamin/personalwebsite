Website Overview

As a firm advocate of simple and subtle websites being the best, this website was built and designed with a simplistic and nature theme. As such, no animations, gif or videos have been included in this website. Inclusion of images has also been limited to a minimal amount as too much images can overwhelm the contents. Thus, this site was designed with a simple and subtle approach where the user will not focus too much on the design than the contents.

Technical

The site is composed mainly of 4 html files and one cascading style sheet in accordance to the requirements of the assignment. The table below provides the overall structure of the site.

File Name File Type Page Name Contents
index.html HTML HOME This page contains a short introduction and a welcome image.
past.html HTML PAST This page contains information of the past experiences.
future.html HTML FUTURE This page contains future aspirations.
comments.html HTML COMMENTS This page contains information on the overview, aesthetics, and technical aspects of the website.
websystems CSS - This file controls the aesthetics aspect of the website

While the contents of each web pages differ to one another, the layout of the 4 html is the same and is made up of 4 main div classes and each main div have a fixed 100% width.

1.header

The header class contains the title of the assignment “Min Thu Ta” quoted in simple <h1> tags. The text has been aligned in the center.

2.navigation-row

The div class consists of the following classes:

  • left: has a nested div class “title” which states the heading of the page the user is on. The class left which have the heading of the page has a width of 40%, padding of top and bottom 10 px and a border radius of 0-300-300-0 px. The text is aligned left.
  • nav-bar: contains navigation links (.link) “HOME”, “PAST”, “Future”, “COMMENTS”. Each link has been assigned to specific hyperlink for navigation. The text in the navigation bar are aligned right. The navigation bar has a width of 60% floated left and has a padding top of 25px.

3.body-row

The body row div, which consists the content of the web page is made up of various tags with various specification( font size, color and type )depending on the content (heading tags <h2><h3>, lists <ul> , images <img> and paragraph tags <p> ) of the page the user is on.

The body-row has a padding top of 50 px.Images have also been allocated into their own div, to adjust and position them on the page using CSS. A clear example can be found on the Home page where the text contents were positioned right and the image at the left.

4.footer

The footer div is also composed of the following classes and all the text in the footer div is aligned center:

  • detail: includes the following text “University of Technology, Sydney “which states what the website is for.
  • author: includes the following text “Website by Min” states the author of the website.

More Information

This website has also be tested on microsoft edge and on android.

Aesthetic

Color

The initial design of the website was a simple black and white approach. But upon completion, the design was too simple and the contrast black text on white was nowhere near subtle. Hence, after several variations, the website was finished with grey (as in the color of stone) on a snow background giving the website less contrast with a sense of hue. Moreover, users will also realize that a taint of green has also been added to the website upon hovering the pointer on the navigation bar.

Font

  • Depending on the contents throughout the website, various font sizes are used.
  • All texts have the same font styles, font color and font weight except for the texts in table, the title and the text in the footer.
  • The texts in the title, footer and table headers are white in color. The table headers are also bold.

Interactivity

  • To illustrates interaction, an underline will appear under the navigation option and the text will turn green upon hovering the pointer on the top navigation bar.

Accessibility

  • The website has been designed and built so that every site visitor can view the contents.
  • All the images have an “alt” attribute that states a general idea of what the image is illustrating.
  • The title, the headings and the navigation are large and clearly placed.
  • The site is composed of grey texts and snow (almost white background) and hence all the contents are clearly illustrated and placed.
